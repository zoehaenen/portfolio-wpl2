# Portfolio — Zoë Haenen

Dit is mijn persoonlijke portfolio, ontwikkeld binnen **Werkplekleren 1**.  
De website bundelt mijn werk als toekomstig front-end developer met een sterke focus op visueel ontwerp, interactie en storytelling.

De portfolio bevat zowel digitale projecten (front-end, UI design) als creatief werk (animatie, analoge fotografie en schetsboeken).

## Live versie

🔗 https://zoehaenen.github.io/portfolio-wpl2/

## Inhoud

- **Home**  
  Editorial landingpagina met vaste header en hero-layout.

- **Projecten**
    - Scripting (HTML, CSS, JavaScript)
    - UI Design (Figma → visueel ontwerp)
    - Fotografie (analoge fotografie met lightbox-galerij)
    - Animatie (video-embed via Vimeo & YouTube)
    - Schetsboek (proces & visueel onderzoek)

  Korte voorstelling, motivatie en focus als designer/developer.

- **Contact**  
  Contactgegevens en externe links.

---

## Belangrijke opmerking — Scripting project

De **live versie van het Scripting-project** is momenteel **niet toegankelijk**.

- De website werd oorspronkelijk gehost op **Netlify**, maar kan niet meer geopend worden omdat de beschikbare Netlify-credits zijn opgebruikt.
- Het project kan voorlopig ook **niet elders gepubliceerd worden**, aangezien de bijhorende repository **privé** staat.

De **volledige broncode** en projectstructuur zijn wel correct uitgewerkt en lokaal beschikbaar.  
De focus van dit project ligt op **JavaScript-logica, DOM-manipulatie en state management**, eerder dan op permanente hosting.

---

## Technische opbouw

- **Vue 3 + Composition API**
- **Vite**
- **Vue Router**
- **Component-based architecture**
    - gedeelde layout en navigatie
    - herbruikbare projectkaarten
    - herbruikbare galerij en lightbox
    - data-driven projectinhoud
- **CSS**
    - scoped component styling waar passend
    - globale basisvariabelen en reset
    - responsive layout (desktop, tablet, mobiel)
- **Externe tools**
    - Adobe Fonts
    - Vimeo & YouTube embeds
